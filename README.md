# 📊 Olist电商数据分析项目



**一个完整的电商数据分析项目，包含数据清洗、用户分群、业务洞察**

## 📊 项目概述

这是一个基于巴西Olist电商数据集的数据分析项目，旨在通过用户行为数据挖掘业务价值，提供可执行的运营建议。

## 数据来源
本项目使用Kaggle上的巴西电商数据集：
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## 需要下载的文件
下载后请将以下文件放入 `data/` 文件夹：

1. `olist_orders_dataset.csv`
2. `olist_order_items_dataset.csv`  
3. `olist_customers_dataset.csv`
4. `olist_products_dataset.csv`
5. `product_category_name_translation.csv`


## 🔍 核心分析内容

- **用户转化路径分析**：构建转化漏斗，识别流失环节

- **RFM用户分群**：使用K-Means聚类识别4类用户群体

- **品类销售分析**：识别明星品类和增长机会

- **业务建议输出**：基于分析结果的实战建议



## 怎么运行？

1. 安装Python和这些包：pandas, matplotlib, sklearn

2. 下载数据放到data文件夹

3. 用Jupyter打开"电商用户行为分析.ipynb"
